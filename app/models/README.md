# 模型定义目录

## 目录定位

`app/models/` 定义 API 请求体与响应体的 Pydantic 模型，是前后端 JSON 契约的唯一模型层。

## 当前文件

1. `api_models.py`：定义筛选任务、维护任务、概念任务、日志流和前端设置保存接口的模型。
2. `__init__.py`：包初始化。

## 当前模型覆盖范围

1. 筛选任务创建、状态、控制、结果、日志与逐股状态模型。
2. 维护任务创建、状态、控制、日志与页面表单设置模型。
3. 概念任务创建、状态、控制与日志模型。
4. 结果股票概念信息模型。

## 维护建议

1. 接口字段名、默认值、校验范围一旦修改，都属于正式接口变更，必须同步更新 README、HANDOFF、路由层和前端。
2. `extra="forbid"` 或嵌套模型的宽松策略属于接口约束的一部分，修改时要写清原因。
3. 若路由新增新的 JSON 结构，应优先先补模型，再补路由，而不是直接返回匿名字典长期漂移。
